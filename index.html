CSS Basics and The Box Model

Using CSS within HTML

CSS is used to style pages and make them appear attractive.

Different ways to implement CSS in a HTML document:

  1. Inline - Within the HTML code as attributes
  2. Internally - Within the HTML document at the head section using 'style' tag
  3. Externally - By linking an external stylesheet to the HTML file using the 'link' tag
